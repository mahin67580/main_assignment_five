Question-1: What are the different ways to select an element in the DOM?
1. document.getElementById("id") – Selects an element by ID.
2. document.getElementsByClassName("class") – Selects elements by class name.
3. document.getElementsByTagName("tag") – Selects elements by tag name.
4. document.querySelector("selector") – Selects the first matching element.
5. document.querySelectorAll("selector") – Selects all matching elements.
Question-2: What is the difference between innerHTML, innerText, and textContent ?
1. innerHTML – Retrieves or sets HTML content, including tags.
2. innerText – Retrieves or sets visible text while ignoring hidden elements.
3. textContent – Retrieves or sets all text, including hidden content, without parsing HTML.
Question-3: What is event delegation in the DOM?
Event delegation is a technique where a parent element handles events for its child elements using event bubbling. This improves efficiency, especially for dynamically added elements.
Question-4: What is event bubbling in the DOM?
Event bubbling is when an event starts from the target element and propagates up through its parent elements in the DOM hierarchy.
Question-5: How do you create, add, and remove elements using JavaScript?
1. Create: document.createElement("tag")
2. Add: parent.appendChild(child)
3. Remove: parent.removeChild(child)
